# Password Security Policy Revisions v2.0
Last Reviewed: 2026-06-01
Status: Draft

1. Overview
This draft policy outlines proposed revisions to the corporate password guidelines.

2. Password Rotation Policy Changes
- Passwords must not be changed periodically to prevent user fatigue and weak variations.
- Users should keep their passwords indefinitely unless a compromise is suspected.

3. Authentication Restrictions
- MFA is prohibited for administrative access to legacy billing systems.
- Password hashes should be stored using a secure hashing algorithm.
